# Available Skill Categories
When assigning tasks to `GenericAgent`, you MUST choose ONE of the following categories that best fits the required action:

* **`web_scraping`**: 
  Use this skill to fetch, read, parse, or scrape text content from a specific URL (e.g., when the user provides a link or after finding a link in a search).
* **`system_ops`**: 
  Use this skill to interact with the host operating system (Windows). This includes executing Windows PowerShell commands, reading local files, managing directories, or checking system hardware/network status.
